O Que é HTML?

HTML (HyperText Markup Language) é uma linguagem de marcação de hipertexto usada para definir a estrutura da página web, atráves dos elementos são feitas as marcações que delimita o conteúdo da página.

As tags são usados para definir o tipo do conteúdo para o navegador. É possível colocar um texto dentro de uma tag de título, mas não é recomendável.


Anatomia das Tags

Tag de abertura: Consiste no nome do elemento envolvido em sinais de maior e menor.

Tag de fechamento: Mesma estrutura da tag de abertura, mas inclui uma barra antes do nome do elemento.

Conteúdo: É o conteúdo do elemento, que no exemplo é um texto.

Elemento: É a estrutura completa como um todo da tag.

Estrutura Básica

  • Elemento html envolve todo o conteúdo da página.

  • Elemento head recebe na página HTML conteúdos que não serão mostrados pro Usuário sendo no máximo um title da página.

  • Elemento meta charset utf-8 inclui todos os caracteres dos idiomas existentes.

  • Elemento title define o titulo que aparece na aba do navegador.

  • Elemento body contém todo o conteúdo mostrado aos usuários, podendo ser imagens, textos, videos entre outros.

Imagens

A tag img integra imagens na página Web, que através do atributo src é direcionado o caminho onde essa imagem está armazenada, podendo ser no próprio projeto com o caminho ou através de links de imagens da web.

O atributo alt especifica textos discritivos para usuários que não conseguem ver a imagem, exemplos:

  1. Devido a deficiencia visual, onde esses usuários usam ferramentas de leitura de tela para ler o texto.
  2. Imagem não carregada, podendo ser causado por conta da imagem não estar armazenado no projeto ou a imagem da web foi retirada de ar.

Títulos e Subtitulos

Tags de título são elementos que possuem níveis de prioridade, onde h1 é o titulo principal da página e de h2 até h6 sendo subtitulos.

Listas

Existem dois tipos comuns de Listas no HTML, que são listas ordenadas e não ordenadas.

  • Listas não ordenadas a ordem dos itens não importa, sendo representado pelo elemento ul.

  • Listas ordenadas a ordem dos itens importa, sendo representado pelo elemento ol.

  • Quando for digitado os itens da lista, cada um desses itens é preciso ser colocado dentro da tag li, tag essa que representa cada um dos itens.
